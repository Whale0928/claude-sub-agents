## ⚡ 설치 방법

### 1. 자동 설치 (권장)

```bash
# 저장소 클론
git clone [repository-url]
cd claude-sub-agents

# 에이전트 설치
chmod +x copy-agents.sh
./copy-agents.sh
```

### 2. 수동 설치

```bash
# 에이전트 디렉토리 생성 (없는 경우)
mkdir -p ~/.claude/agents

# 파일 복사
cp agents/* ~/.claude/agents/
```
